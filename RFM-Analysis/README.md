# Customer Segmentation Using RFM Analysis
# Overview of the Project
The project performs Customer Segmentation using the RFM(Recency, Frequency, Monetary) analysis.
# What is RFM?
- Recency: No. of days since the last purchase made.
- Frequency: Number of purchases done by the customer.
- Monetary: What is the total spending of the customer.
# Steps:
1. Data Preprocessing
2. Feature Engineering
3. RFM Calculation
4. Scoring using Quartiles
5. Customer Segmentation
6. Data Visualization
# Segmentations of Customer:
- VIP
- Loyal
- At Risk
- Regular
# Tools Used
- Python
- Pandas
- Maltplotlib
# Output
Created a bar chart to visualize the distribution of customer segments
# Conclusion:
RFM analysis helps identify valuable customers and improve marketing strategies.
